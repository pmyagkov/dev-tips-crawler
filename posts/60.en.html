DevTools: Edit HTML in the Console Panel of DevTools ===== https://habrastorage.org/files/468/431/6d8/4684316d8a254c618c8a0d8885541457.gif =====

The DOM node context menu you may be familiar with from the Elements panel is present in the Console Panel also.

  1. Log a DOM node to the console.
  2. Right click on it.
  3. Select Edit as HTML or Edit Text.
  4. Notice the DOM is updated on the page and also the Elements Panel.
